Privacy Policy

Your privacy is important to us. This policy explains how we handle your information.

Information We Collect Minimal Data

SpendSnap is a student project designed to help you track expenses. We only collect essential information:

  • Account details (email and password)
  • Expense information you enter
  • Basic usage statistics to improve the service

Note: We do not collect sensitive financial information like bank account numbers or credit card details.

Data Security Spring Boot Security

We take your privacy seriously and implement strong security measures:

  • All passwords are encrypted using BCrypt encryption
  • Spring Security framework protects against common vulnerabilities
  • Data is stored securely with protection against SQL injection attacks
  • Regular security updates and monitoring

No Third-Party Sharing Student Project

Since this is a student project and not a commercial application:

  • We do not share your data with any third parties
  • We do not sell your personal information
  • We do not use your data for advertising purposes

Your expense data is only used to provide the tracking functionality within the app.

Data Retention

Your data is stored for as long as your account remains active. You can:

  • View all your stored information through the app interface
  • Export your expense data at any time
  • Delete your account and all associated data permanently

Educational Purpose

Please remember that SpendSnap is a student project created for educational purposes:

  • It demonstrates web development skills using Spring Boot
  • It showcases modern security practices
  • It's not intended for commercial use

While we implement strong security measures, please avoid entering highly sensitive information as this is a student project environment.

This Privacy Policy was last updated on January 1, 2025.
